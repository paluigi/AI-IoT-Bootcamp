Software Requirements:
----------------------
*	Python 2.7.3 (Developed on 64 bit)
*	MS SQL Server 2012 (SentiWordNet stored in SQL Server)

Python Libraries
----------------
*	pyodbc (For accessing SQL Server Database)
*	inflect (For generating plurals,singular nouns etc.)
*	xlrd (For reading MS Excel Sheet)
*	collections (General purpose built-in containers)
*	Natural Language Toolkit (NLTK)

Files
-----
*	Machine Readable English Dictionary
*	Lexicon of Positive and Negative Words
*	List of Positive and Negative Emoticons

Others
------
List of Negations, Intensifiers and Contextual Shifters
